## 附录

- 注意：上课的时候突然卡壳了！`readline()` 在遇到行尾的时候返回空白字符串，转成`bool`型是`False`，正确的按行读取并且加载到数组中应该是这样：

  ```python
  array = []
  with open("test.txt", "r", encoding="utf-8") as f:
      text = f.readline()
      array.append(text)
      while text:
          text = f.readline()
          array.append(text)
  print(array)
  ```
