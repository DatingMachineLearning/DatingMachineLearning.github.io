## 课后作业

1. 编写一个程序：能顺序地输入两个数，回车后，程序将在在屏幕上打印两个数分别**加减乘除**的结果，程序代码应该包括**你的注释**。
2. 编写一个程序：能先输入字符串 s，然后输入一个整数 n，程序将把 **n 个字符串拼接**打印出来。
